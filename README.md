# 📈 StockinShip - Real-Time Stock Analytics Dashboard

## 🚀 Project Overview

StockinShip is a real-time stock market analytics dashboard built using Python and Streamlit.

The project allows users to:
- Analyze live stock market data
- Compare multiple companies
- Visualize data using different graphs
- Convert stock prices into multiple currencies
- View AI-based stock predictions
- Analyze technical indicators

This project combines:
- Data Science
- DBMS Concepts
- Machine Learning
- Financial Analytics
- Data Visualization

---

# ✨ Features

## 📊 Real-Time Stock Data
Fetches live stock market data using Yahoo Finance API.

## 📈 Multiple Graph Visualizations
Supports:
- Line Chart
- Bar Chart
- Area Chart
- Scatter Plot
- Candlestick Chart

## 🔥 Multi-Company Comparison
Compare multiple companies simultaneously.

## 💱 Currency Conversion
Convert stock prices into:
- USD
- INR
- EUR
- GBP
- JPY

## 🤖 AI Stock Prediction
Uses Linear Regression to predict future stock prices.

## 📉 Technical Indicators
Includes:
- Moving Average 50
- Moving Average 200
